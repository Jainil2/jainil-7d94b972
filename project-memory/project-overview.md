# Project Overview
**Name:** Stateful AI Portfolio
**Type:** Frontend-Heavy Personal Portfolio & Engineering Showcase
**Tech Stack:** React, Vite, Tailwind CSS, Zustand

**Core Objectives:**
1. Showcase personal portfolio and resume data.
2. Build interactive, high-fidelity visualizations of System Design and DSA concepts.
3. Maintain a scalable frontend architecture.
4. Demonstrate knowledge of Distributed Systems and Security.

**Agent Roles:**
- **Frontend Engineer:** UI/UX, components, routing, Tailwind styling.
- **Solution Architect:** System design, DSA visualizations, architectural patterns, security.
- **AI Project Manager:** Prompt tracking, memory persistence, state consistency.
